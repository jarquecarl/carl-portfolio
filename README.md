# 🖥️ Developer Portfolio

## Sections

| Section | Content |
|---|---|
| Projects | Featured work with live demo and repo links |
| Skills | Tech stack organized by category |
| Experience | Internship and work history |
| Certifications | All certificates with verifiable images |
| About | Background, education, and interests |
| Contact | Direct contact form |

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + TypeScript |
| UI Components | shadcn/ui |

## Getting Started

```bash
git clone https://github.com/jarquecarl-debug/carl-portfolio.git
cd carl-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Deploy to Netlify

```bash
npm run build
netlify deploy --dir=dist --prod
```
